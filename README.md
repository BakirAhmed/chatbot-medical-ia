#  Chatbot Médical - Diagnostic par IA

## Description

Chatbot médical utilisant le Machine Learning (Random Forest) pour prédire une maladie à partir des symptômes décrits par l'utilisateur. Le chatbot affiche également la sévérité des symptômes, une description de la maladie et 3 précautions.

## Fonctionnalités

- Détection des symptômes en langage naturel (français/anglais)
- Affichage de la sévérité de chaque symptôme (1 à 10)
- Prédiction de la maladie (Random Forest)
- Description de la maladie
- 3 précautions recommandées
